# 📚 Sistema de Acompanhamento Escolar – MVP

Projeto desenvolvido para a disciplina **Projeto Integrado II** do curso de **Análise e Desenvolvimento de Sistemas – UFCA (CEAD)**.

O objetivo do projeto é desenvolver a **prototipação de um sistema educacional** que permita o acompanhamento do desempenho escolar dos alunos por professores, responsáveis e administradores.

---

# 🎯 Objetivo do Projeto

Desenvolver a **prototipação de um MVP (Minimum Viable Product)** de um sistema que permita:

- Registro de notas
- Registro de frequência
- Acompanhamento do desempenho escolar
- Visualização de relatórios acadêmicos

O sistema busca **facilitar a comunicação entre escola, professores e responsáveis**, oferecendo uma plataforma simples e organizada para o acompanhamento da vida escolar do aluno.

---

# 🧩 Escopo do MVP

O MVP do sistema contempla três tipos principais de usuários:

### 👨‍🏫 Professor
- Registrar notas
- Registrar frequência
- Visualizar turmas
- Acompanhar desempenho dos alunos

### 👨‍👩‍👧 Responsável
- Visualizar notas do aluno
- Visualizar frequência
- Acompanhar desempenho escolar

### ⚙️ Administrador
- Gerenciar usuários
- Gerenciar turmas
- Gerenciar alunos
- Gerenciar disciplinas
